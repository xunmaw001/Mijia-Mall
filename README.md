# Mijia-Mall
基于Spring Boot和Vue.js的米家商城是一款专注于智能家居产品在线销售的电商平台。后端采用Spring Boot架构，提供稳定的业务逻辑处理，前端通过Vue.js实现直观且互动性强的用户界面。用户可以轻松浏览米家系列产品，进行在线购买、支付和订单管理，享受一站式购物体验。此外，该平台还支持产品评价和用户互动，促进消费者对智能家居产品的了解与购买决策。
